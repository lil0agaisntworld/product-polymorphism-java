# Product Polymorphism (Java)

Simple Java console application demonstrating OOP concepts:
- Inheritance
- Polymorphism
- Method overriding
- Enums

## About
This project simulates a product system where different product types 
(common, used, imported) generate customized price tags using polymorphism.

## Example Output
PRICE TAGS:
Notebook $ 1100.00
Tablet $ 400.00 (Customs fee: 20.00)
iPhone (used) $ 400.00 (Manufacture date: 15/03/2017)
